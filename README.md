# Mochi

Mochi is a cozy 3D desktop companion that lives in a small interactive room and can visit your desktop as a lightweight companion window.

## Features

- Care for Mochi through gentle room interactions.
- Feed, play, comfort, and let Mochi rest.
- Watch the room change as Mochi learns your routines.
- Garden growth continues while Mochi is open, including when the window is minimized.
- Closed sessions do not simulate missed time.
- Enable a transparent desktop companion moment during play.
- Local save data keeps progress on your machine.

## Development

```bash
npm install
npm run build
npm start
```

## Release

```bash
npm run dist
```

The Windows installer is generated in `release/`.
